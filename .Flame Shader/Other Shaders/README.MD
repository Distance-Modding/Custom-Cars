# Other Shaders

Newer shaders that have been toyed arround with and done successfully will be added here
key thing to keep in mind, these shaders need the donotreplace material in order for them to work

It is also important to not rename the file otherwise the script will get messed up

Remember, Custom Shaders only work with the donotreplace material

<details>
  
  <summary>Shaders</summary>
  
  * Camera Shader - attach this shader to a material to attach to a camera to see out of an invisible camera attached to the car
  * Emission Pulse Shader - attach this shader to a material to allow a pulsing emission effect with speed adjustments
  * Transparent Emission - Used mostly for Trail Renders to give off a transparent light effect
  * Sprite Emission Shader - Helps with Sprite Trail Renders
  * Boost Flame Trail - A Trail Render Shader that allows Sprites to act like a flame
  * Custom Flag Effect - Turns the trail into a flag, adjustable if you don't want the flag animation. Kinda buggy, video coming soon
  * Tread Motion - A Standard Surface Shader that allows cars to have functional Tank Treads
  * Window Shader - Adds a realistic reflection shader if your car does not have an interior equipped with an RGB System for different tint color
  * Transparent Emission - Allows materials to be transparent but still have an emission
